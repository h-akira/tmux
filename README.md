# Introduction
```
cd
git clone --recursive git@github.com:h-akira/tmux.git ~/.tmux
ln -sf .tmux/dot.tmux.conf .tmux.conf
tmux source-file ~/.tmux.conf
```
After that, while running tmux, `prefix + I`.
